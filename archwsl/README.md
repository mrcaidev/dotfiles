# Arch WSL

Bootstrap Arch Linux on WSL.

## 🚀 Getting Started

1. Download AppX and certificate from [ArchWSL's releases](https://github.com/yuk7/ArchWSL/releases).
2. Install certicate to "Trusted People" on local machine.
3. Double click AppX and install the distro.
4. Run `personalization.sh` in Arch WSL.
5. Run `config.ps1` in PowerShell.
